Definitions

variables are declared with an '=' statement.
The RHS can contain math, numbers and other variables
<variable name> = <number>
MOTfrequency = 12 +4
Tstart = 2
MOTdur = 10 + Tstart

If one of the pulse keywords is used as a variable name, the syntax will alert you:
Freq = 10 + Tstart

If one of the Parameter Vault variables are used, it will also light up:
Tstart = A

A = 4

Pulses

All pulse declarations follow a simple pattern:

DDS_2: freq 37 MHz, amp -10 dBm, at 350 ms, for 20 ms, mode Normal

DDS_2: is the channel name

freq, amp, at, for (fromfreq, fromamp): are keywords that must be followed by a number (or a variable) and a unit.
The keywords in parathesis can be omitted, but the others must always be specified.

mode is the only keyword that is not followed by a number and a unit. It is followed by either Normal or Modulation,
specifiying either normal pulse mode, or frequency modulation mode.

 

 

<ChannelName>: freq <num> <unit>, amp <num><unit>
